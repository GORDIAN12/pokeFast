# Pokémon API - FastAPI

API RESTful para gestionar pokemones y sus evoluciones.

## Stack:
- FastAPI + SQLModel + SQLite
- Alembic para migraciones
- Pydantic para validaciones
- CRUD completo
- Filtros por tipo y búsqueda por nombre

## Endpoints:
- `GET /pokemons`
- `GET /pokemons/{id}`
- `POST /pokemons`
